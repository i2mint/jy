Get (name, params) pairs of function definitions extracted from js_code.

>>> js_code = '''
... function foo(a, b="hello", c= 3) {
...     return a + b.length * c
... }
... const bar = (y, z = 1) => y * z

...         {'name': 'y'},
...         {'name': 'z', 'default': 1}
...     ]),
...     ('func.assigned.to.nested.prop', [
...         {'name': 'x'}
...     ])

jy.js_parse.parse_js(js_code: str, encoding: str | None = None, *, ingress=<function <lambda>>, egress=<function <lambda>>) → Script[source]

Parse JavaScript source (or a path to a .js file) into an esprima AST.

js_code may be a string of JS or a filesystem path, in which case the file is read (honoring encoding). ingress preprocesses the source before parsing; egress post-processes the resulting AST.

>>> tree = parse_js("const x = 42;")
>>> tree.body[0].type
'VariableDeclaration'
>>> tree.body[0].declarations[0].id.name
'x'

jy.js_parse.replace_ids_in_code(code: str, patterns: Tuple[str] = ('id="([^"]+)"', '#([a-zA-Z][\\w\\-]*)', 'document\\.getElementById\\("([^"]+)"\\)')) -> (<class 'str'>, <class 'dict'>)[source]

Replaces IDs in a mixed HTML, CSS, and JS string with unique versions.

Args: - code (str): The input code containing HTML, CSS, and/or JS. - patterns (tuple): A tuple of regex patterns to match IDs.

Returns: - tuple: A tuple containing the modified code and a dictionary of old-to-new ID mappings.

This function is useful to make id-unique copies of HTML/JS components that have IDs. This need was noticed and discussed in: https://github.com/i2mint/jy/discussions/3#discussioncomment-6862843

jy.js_parse.variable_declarations_pairs(src, *, encoding=None)[source]

Yield (name, value) pairs for the top-level variable declarations in JS.

Literal, object, array and unary-minus initializers are evaluated to their Python equivalents; other initializer kinds yield None. A leading export on a declaration is stripped first, so ES-module sources parse.

>>> list(variable_declarations_pairs("const a = 3; let b = 'hi';"))
[('a', 3), ('b', 'hi')]
>>> list(variable_declarations_pairs("export const config = {x: 1, y: [2, 3]};"))
[('config', {'x': 1, 'y': [2, 3]})]
>>> list(variable_declarations_pairs("var n = -5; const flag = true;"))
[('n', -5), ('flag', True)]
